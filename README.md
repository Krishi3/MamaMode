# MamaMode

MamaMode is an AI-powered postpartum support system that provides emotional guidance and resources to new mothers through SMS and voice interactions. Instead of requiring an app download or login, users can text or call to receive immediate support when they may feel overwhelmed, stressed or isolated.

The system integrates AI-driven conversation, structured resource retrieval, and voice synthesis to deliver accessible, safety focused support anytime it’s needed.

Authors: Shaivi Sahni, Prachi Patel, Krishi Dalal

Tech Stack:
- Google Gemini API
- ElevenLabs API
- Snowflake API
- Twilio API

Languages: TypeScript, SQL
Database: MongoDB
